GT On-Campus Jobs

more than 300 users as of Jan 16 2020

The Problem:

Most on-campus jobs at Georgia Tech are posted on this sttaic webiste. Students have to check the website for updates manually.Vacancies get filled quickly, and there's no way to get notified as soon as vacancies open.

The Solution:

We built an email notification service that sends an email as soon as a vacancy is posted. Users subscribe using their email and get an email notification as soon as a vacancy is posted.

How does it work?

  1. Scrapes Georgia Tech on-campus job board using Beautiful Soup API
  2. Compares newly found job postings to existing jobs postings stored in remote MongoDB
  3. If a new posting if found (compares job descriptions to check if a posting is new), trigger a notification
  4. Sends notification using GMAIL API. User emails are stored in Mailchimp
